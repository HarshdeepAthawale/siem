# MongoDB Utilities

This folder contains MongoDB-related scripts, utilities, and configurations.

## Structure

- `scripts/` - Database initialization and utility scripts
- `seeds/` - Seed data for development/testing
- `migrations/` - Database migration scripts (if needed)

## Usage

### Initialize Database

```bash
node mongodb/scripts/init.js
```

### Seed Sample Data

```bash
node mongodb/scripts/seed.js
```
